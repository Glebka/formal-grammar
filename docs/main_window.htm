Главное окно

Главное окно Formal Grammar состоит из четырех частей:

- Строка меню  - Панель инструментов - Поле ввода текста грамматики - Список ошибок и предупреждений

В строке меню доступны все функции текстового редактора: создание, открытие, сохранение, печать текста грамматики, поиск и замена текста.

Панель инструментов содержит такие кнопки:

 - создать новый файл грамматики

 - сохранить изменения в файл

 - включить/выключить проверку бесконечной рекурсивности правил

 - построить матрицу простого предшествования
