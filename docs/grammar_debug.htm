Отладка грамматик

Под отладкой грамматики понимают процесс преобразования заданной грамматики с целью удовлетворения требований, которые предъявляет к ней используемый метод синтаксического анализа. Для того чтобы разрешить (исправить) конфликты (коллизии) матрицы предшествования, необходимо преобразовать правило исходной грамматики таким образом чтобы исключить конфликтные отношения. При этом всегда исходная и модифицированная грамматики должны быть эквивалентны. Для того чтобы исправить конфликты типа "<=" и ">=", удобно доопределить грамматику таким образом, чтобы убрать отношение "=".

Процесс исправления конфликтов выглядит следующим образом:

  1. Вычисляется матрица предшествования (или её фрагмент)
  2. Конфликты интерпретируются с помощью синтаксического дерева.
  3. Грамматика анализируется и модифицируется
  4. Пункты повторяются до тех пор, пока в матрице предшествования есть неразрешённые конфликты.
