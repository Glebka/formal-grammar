Матрица простого предшествования

Окно матрицы простого предшествования позволяет удобно просматривать отношения между парами символов. Если грамматика имеет большое количество терминальных и нетерминальных символов, то отношения между ними лучше просмтаривать с помощью двух выпадающих списков "Символ А" и "Символ Б". Соответствующая ячейка матрицы будет подсвечена, а само отношение будет показано между выпадающими списками. В окне также можно просмотреть список конфликтов. Щелчек по элементу из списка подсветит соответствующую ячейку в матрице. Вы можете сохранить матрицу в формате HTML с помощью кнопки
