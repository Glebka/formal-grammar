Метод синтаксического анализа простого предшествования

Если задана цепочка # x1 x2 x3… # то для определения основы необходимо продвигаться по цепочке слева на право и определять отношения между соседними символами до тех пор, пока не встретилось отношение ">", тогда выполняется возврат назад по цепочке до ближайшего отношения "<" . Основой является цепочка, которая окружена с двух сторон отношениями "<" ">" . Данную основу необходимо сравнить с правыми частями правил и заменить её на левую часть правила. Процесс продолжается до тех пор, пока цепочка #S# не будет свёрнута в начальный символ.

Как работает метод синтаксического анализа?
